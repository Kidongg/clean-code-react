# 1. 중괄호(Curly Braces)를 사용하는 방법으로는 무엇이 있을까?

- 문자열일 경우 중괄호를 생략할 수 있다.
- 자바스크립트 로직이 필요한 경우 중괄호를 사용해야 한다.
- 더블 중괄호는 중괄호안에 객체를 넣은 것이다. 아래 코드는 똑같이 동작한다.

```
const Component = () => {
    const styles = {
        backgroundColor: "blue",
        width: 100
    }

    return (
        <button
            className="clean-code"
            style={styles}
            style={{
                backgroundColor: "blue",
                width: 100
            }}
        />
    )
}
```
